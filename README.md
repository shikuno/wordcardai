# スマート単語カード

**AIがあなたの代わりに単語カードを作ってくれる、かんたん学習アプリ**

[![App Store](https://img.shields.io/badge/App%20Store-Download-blue?logo=apple)](https://apps.apple.com/app/id0000000000)

---

## アプリについて

「スマート単語カード」は、単語カードづくりをまるごとおまかせできる学習アプリです。  
単語を入力するだけで、AIが意味・品詞・例文を自動でカード化してくれるので、あなたは「覚えること」だけに集中できます。

### 主な機能

- 🤖 **AIおまかせ単語カード作成** — 入力するだけで自動生成
- 📚 **コレクション管理** — テーマ・目的ごとにカードを整理
- 🎴 **学習モード** — フラッシュカード形式でテンポよく復習
- 🔒 **完全オフライン対応** — データは端末内に安全に保存
- 🌓 **ダークモード対応** — システム設定に自動追従

## サポート

### よくある質問（FAQ）

**Q. アプリが起動しない／クラッシュする**  
A. 一度アプリを完全に終了し、再起動してください。それでも解決しない場合は下記までお問い合わせください。

**Q. 単語カードが保存されない**  
A. 端末のストレージ空き容量をご確認ください。空き容量が少ない場合、保存に失敗することがあります。

**Q. AIの翻訳結果がおかしい**  
A. AIによる自動生成のため、稀に不正確な結果が出ることがあります。カード編集画面から手動で修正できます。

**Q. データのバックアップはできますか？**  
A. 現在は端末内保存のみです。iCloud 同期機能は今後のアップデートで対応予定です。

**Q. アプリを削除するとデータはどうなりますか？**  
A. アプリを削除するとすべてのデータも削除されます。削除前にご注意ください。

## プライバシー

- このアプリはユーザーの個人情報を収集・送信しません。
- すべてのデータは端末内にのみ保存されます。
- インターネット接続は AI 翻訳機能の利用時のみ使用します。

詳細は [プライバシーポリシー](PRIVACY_POLICY.md) をご確認ください。

## 開発者向け情報

### セットアップ

```bash
git clone https://github.com/shikuno/wordcardai.git
cd wordcardai
open WordCardAI.xcodeproj
```

### プロジェクト構成

```
WordCardAI/
├── Models/          # データモデル
├── ViewModels/      # ビジネスロジック（MVVM）
├── Views/           # UIコンポーネント
├── Services/        # AI翻訳・ストレージサービス
└── Utilities/       # ユーティリティ・拡張
```
