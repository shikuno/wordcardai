# プライバシーポリシー / Privacy Policy

**最終更新日: 2025年11月23日**

## 日本語

### はじめに

Yuya Furuichi（以下「開発者」）は、WordCardAI（以下「本アプリ」）をプライバシーを重視したアプリとして開発しました。本プライバシーポリシーは、本アプリがどのようにユーザー情報を取り扱うかを説明するものです。

### 情報収集について

**本アプリは、一切の個人情報を収集しません。**

本アプリは以下のような情報を収集、保存、送信しません：

- 個人を特定できる情報（名前、メールアドレス、電話番号など）
- 位置情報
- デバイス情報
- 使用状況データ
- クラッシュログ
- 分析データ

### データの保存

本アプリで作成されたすべてのデータ（カード集、単語カード、設定など）は、**ユーザーのデバイス内にのみ保存**されます。

- データは iOS の UserDefaults を使用してローカルに保存されます
- データは外部サーバーに送信されません
- データは第三者と共有されません
- アプリをアンインストールするとデータは完全に削除されます

### AI 翻訳機能について

現在のバージョン（1.0.0）では、AI 翻訳機能はモックサービスとして実装されており、**実際のネットワーク通信は行いません**。

将来のバージョンで Apple の Translation Framework を統合する予定ですが、その場合でも：

- 翻訳処理はデバイス上で完全にオフラインで実行されます
- 翻訳データが外部サーバーに送信されることはありません

### サードパーティサービス

本アプリは、以下のサードパーティサービスを使用しません：

- 広告ネットワーク
- 分析ツール（Google Analytics など）
- クラッシュレポートツール
- ソーシャルメディア SDK
- その他の追跡ツール

### インターネット接続

本アプリは**完全にオフラインで動作**します。インターネット接続は不要です。

### 子供のプライバシー

本アプリは13歳未満の子供を含む、すべての年齢のユーザーが安全に使用できます。個人情報を収集しないため、COPPA（Children's Online Privacy Protection Act）への対応が不要です。

### セキュリティ

ユーザーデータはデバイス内にのみ保存されるため、iOS のセキュリティメカニズムによって保護されます。開発者はユーザーデータにアクセスできません。

### プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。変更があった場合は、本ページで通知いたします。

### お問い合わせ

プライバシーに関するご質問やご懸念がある場合は、以下までご連絡ください：

- GitHub Issues: https://github.com/yourusername/WordCardAI/issues

## English

### Introduction

Yuya Furuichi (hereinafter "Developer") has developed WordCardAI (hereinafter "the App") as a privacy-focused application. This Privacy Policy explains how the App handles user information.

### Information Collection

**The App does not collect any personal information.**

The App does not collect, store, or transmit the following types of information:

- Personally identifiable information (name, email address, phone number, etc.)
- Location data
- Device information
- Usage data
- Crash logs
- Analytics data

### Data Storage

All data created in the App (card collections, word cards, settings, etc.) is **stored only on the user's device**.

- Data is stored locally using iOS UserDefaults
- Data is not transmitted to external servers
- Data is not shared with third parties
- Data is completely deleted when the app is uninstalled

### AI Translation Feature

The current version (1.0.0) implements the AI translation feature as a mock service and **does not perform actual network communication**.

We plan to integrate Apple's Translation Framework in future versions, but even then:

- Translation processing will be performed entirely offline on the device
- Translation data will not be sent to external servers

### Third-Party Services

The App does not use the following third-party services:

- Advertising networks
- Analytics tools (such as Google Analytics)
- Crash reporting tools
- Social media SDKs
- Other tracking tools

### Internet Connection

The App **operates completely offline**. No internet connection is required.

### Children's Privacy

The App is safe for users of all ages, including children under 13. Since we do not collect personal information, COPPA (Children's Online Privacy Protection Act) compliance is not required.

### Security

User data is stored only on the device and is protected by iOS security mechanisms. The developer cannot access user data.

### Changes to This Privacy Policy

This Privacy Policy may be updated as necessary. When changes occur, we will notify users on this page.

### Contact Us

If you have any questions or concerns about privacy, please contact us:

- GitHub Issues: https://github.com/yourusername/WordCardAI/issues
- Email: your.email@example.com
