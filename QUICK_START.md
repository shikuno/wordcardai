# 🚀 クイックスタートガイド

このガイドに従えば、**5分でアプリをビルドして実行**できます。

## 前提条件

- ✅ macOS がインストールされている
- ✅ Xcode 15.0 以降がインストールされている
- ✅ このリポジトリをクローン済み

## ステップ 1: Xcode でプロジェクトを開く

```bash
cd /Users/yuya/develop/WordCardAI
open WordCardAI.xcodeproj
```

または、Finder で `WordCardAI.xcodeproj` をダブルクリック

## ステップ 2: ファイルをプロジェクトに追加

### 方法 A: 自動追加（推奨）

1. Xcode のプロジェクトナビゲーターで `WordCardAI` グループを選択
2. メニューから **File > Add Files to "WordCardAI"...** を選択
3. 以下のフォルダを選択（⌘キーを押しながら複数選択）:
   - `Models`
   - `ViewModels`
   - `Views`
   - `Services`
   - `Utilities`

4. 追加オプション:
   - ☑️ **Create groups** を選択
   - ☑️ **Add to targets: WordCardAI** にチェック
   - **"Add"** をクリック

### 方法 B: 手動追加

詳細な手順は [XCODE_SETUP.md](docs/XCODE_SETUP.md) を参照

## ステップ 3: ビルド

**ショートカット**: `⌘ + B`

または、メニューから **Product > Build**

### ⚠️ ビルドエラーが出た場合

#### "Cannot find 'CollectionsListView' in scope" エラー
→ ファイルがプロジェクトに追加されていません。ステップ2を再確認してください。

#### "Target Membership" が不正
1. エラーが出ているファイルを選択
2. 右側の File Inspector を開く
3. **Target Membership** で `WordCardAI` にチェック

#### それでも解決しない場合
```bash
# クリーンビルド
⌘ + Shift + K
# 再度ビルド
⌘ + B
```

## ステップ 4: 実行

**ショートカット**: `⌘ + R`

または、メニューから **Product > Run**

### シミュレーターの選択

ツールバー上部で以下のいずれかを選択:
- iPhone 15
- iPhone 15 Pro
- iPhone 15 Pro Max
- その他お好みのシミュレーター

## ステップ 5: アプリを試してみる

### 5-1. カード集を作成

1. アプリが起動したら、右下の **+** ボタンをタップ
2. 「日常会話」と入力
3. **作成** をタップ

### 5-2. 単語カードを追加

1. 作成したカード集をタップ
2. 右下の **+** ボタンをタップ
3. 日本語欄に「おはよう」と入力
4. **💡 候補を生成** ボタンをタップ
5. AI が生成した候補から選択（例: "Good morning"）
6. **保存** をタップ

### 5-3. 学習モード

1. **学習モード** ボタンをタップ
2. 日本語が表示される
3. **答えを見る** をタップして英語を表示
4. **次へ** で次のカードへ

### 5-4. 検索

1. 検索バーに「おはよう」と入力
2. リアルタイムでフィルタリングされる

## 完了！🎉

おめでとうございます！WordCardAI が動作しています。

### 実機でテスト

1. iPhone/iPad を Mac に接続
2. Xcode のデバイス選択で実機を選択
3. `⌘ + R` で実行

## トラブルシューティング

### Q: "No such module 'SwiftUI'" エラー
**A**: Xcode を再起動してみてください

### Q: アプリが起動しない
**A**: シミュレーターを再起動してみてください
```
Device > Erase All Content and Settings...
```

### Q: データが消えた
**A**: シミュレーターをリセットするとデータは消えます。実機では保持されます。

### Q: ビルドが遅い
**A**: 初回ビルドは時間がかかります（1-2分）。2回目以降は高速になります。
