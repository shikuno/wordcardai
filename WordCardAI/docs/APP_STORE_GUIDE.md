このドキュメントは、WordCardAI を App Store に申請するための準備手順を説明します。

- [x] すべての主要機能が動作する

- [ ] アプリアイコン（1024x1024）
- [ ] スクリーンショット（必須サイズ）

## 2. アプリアイコンの準備

### 必要なサイズ

  - 60x60px (@1x)
- **Settings**: 
  - 87x87px (@3x)
  - 58x58px (@2x)
  - 29x29px (@1x)
- **Spotlight**:
  - 120x120px (@3x)
  - 80x80px (@2x)
  - 40x40px (@1x)

- シンプルで認識しやすいデザイン

## 8. ビルドとアップロード

### 1. バージョン番号の確認

Xcode でプロジェクトを開き、Target > General で:
- Version: `1.0.0`
- Build: `1`

### 2. Archive の作成

```
1. Product > Archive
2. アーカイブが完成したら Organizer が開く
3. "Distribute App" をクリック
4. "App Store Connect" を選択
5. "Upload" を選択
6. 署名とプロビジョニングプロファイルを確認
7. "Upload" をクリック
```

### 3. App Store Connect で確認

- アップロード後、10-20分で App Store Connect に表示される
- ビルドを選択してテストフライトでテスト

## 9. TestFlight テスト

### 内部テスト

- 最大100人のテスター
- 審査なしですぐにテスト開始

### 外部テスト

- 最大10,000人のテスター
- Beta App Review が必要（1-2日）

### テスト項目

- [ ] すべての機能が正常に動作
- [ ] クラッシュなし
- [ ] UI の不具合なし
- [ ] パフォーマンス問題なし
- [ ] 各種デバイスでテスト（iPhone SE, iPhone 15 など）

## 10. 審査申請

### チェックリスト

- [ ] アプリアイコン追加
- [ ] スクリーンショット追加
- [ ] 説明文入力
- [ ] プライバシーポリシーURL設定
- [ ] サポートURL設定
- [ ] カテゴリ選択
- [ ] キーワード設定
- [ ] レビューノート記入
- [ ] ビルドを選択
- [ ] 価格設定（無料）

### 申請

1. App Store Connect で "Submit for Review" をクリック
2. 質問に回答
   - Export Compliance: No（暗号化なし）
   - Content Rights: Yes（コンテンツの権利を保有）
3. "Submit" をクリック

### 審査期間

- 通常 1-3 日
- 初回は長くなることもある（1週間程度）

### 審査結果

#### Approved（承認）
おめでとうございます！アプリがApp Storeに公開されます。

#### Rejected（却下）
- 理由を確認
- 問題を修正
- 新しいビルドをアップロード
- 再申請
