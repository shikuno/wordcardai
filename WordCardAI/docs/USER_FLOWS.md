# User Flows (主要フロー)

## 画面一覧
- 単語一覧（List）
- 単語追加/編集（Form）
- 学習モード（Flashcard）
- 設定（バックアップ/復元、同期）

## フロー: 単語の追加
1. 単語一覧で「+」をタップ
2. 単語追加画面で `term` と `meaning` を入力
3. 追加ボタンで保存、一覧に反映
4. バリデーション: `term` は必須

## フロー: 学習モード
1. 学習モード開始を選択
2. ランダム（またはタグ指定）で単語表示（表面: term）
3. 「答えを見る」をタップで meaning を表示
4. 正解/不正解をユーザが選択（後で統計に反映可能）

## フロー: エクスポート
1. 設定→エクスポートを選択
2. JSON/CSV を選びエクスポート（ファイル共有/メールなど）
3. インポート時に重複対応（スキップ/上書き）を選択可能
