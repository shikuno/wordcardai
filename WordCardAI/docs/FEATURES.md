# Features (機能一覧)

## Must-have (MVP)
- 単語一覧表示（リスト）
- 単語追加（term, meaning）
- 単語編集・削除
- ローカル永続化（アプリ再起動後もデータが残る）
- シンプルな学習モード（フラッシュカード）
- 検索（term/meaning）

## Should-have
- タグ/カテゴリ機能（フィルタ）
- 並べ替え（作成日・アルファベット・覚えた順）
- インポート/エクスポート（CSV/JSON）
- 簡易統計（学習回数・登録数）
- バックアップ/復元（iCloud/ファイル）

## Nice-to-have
- クラウド同期（CloudKit/Firebase）
- SRS（間隔反復）学習アルゴリズム
- 発音/音声再生/例文表示
- 共有機能（単語帳の公開・共有）
- 広告/プレミアム課金（IAP）
